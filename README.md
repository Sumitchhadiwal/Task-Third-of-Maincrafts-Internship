# Task-Third-of-Maincrafts-Internship

This repository contains the complete solution for **Artificial Intelligence & Machine Learning – Task 3**, focused on **model validation, overfitting control, and hyperparameter tuning** using Python and scikit-learn.

The project uses the **California Housing Dataset** to build and evaluate regression models for house price prediction.

---

## 📌 Objective

The main objectives of this task are:
- To detect and analyze overfitting in machine learning models
- To apply cross-validation for reliable model evaluation
- To perform hyperparameter tuning using GridSearchCV
- To compare multiple regression models
- To justify the final model selection scientifically
